# Rediscovery Loops Are Workflow Debt

## Observation

One hidden cost of AI-assisted work is repeated rediscovery. The same
constraints, decisions, and boundaries have to be restated before useful work
can continue.

## Pattern

Rediscovery compounds when the workflow does not preserve why a decision was
made, what changed after it, and which actions remain blocked.

## Principle

Operational memory should reduce repeated explanation without turning memory
into automatic permission.
